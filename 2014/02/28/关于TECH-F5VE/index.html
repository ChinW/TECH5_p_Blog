关于TECH F5VE

Feb 28, 2014

TECH F5VE 是由一群热爱计算机技术与艺术设计的学生自主发起的，不隶属于任何组织／部门／机构的技术向组织，其中发起人为Chi。

取意

TECH 取意 Technology (技术), F5VE 取自在Windows中经常使用用的刷新快捷键，总意为“技术与刷新”，意即希望我们能够虚心且不断进取。

原因

TECH F5VE的建立诱因是因当时在学校环境中未能发现非常纯粹的技术讨论圈，Chi因此异想天开尝试自己去建立一个，并非常幸运的得到了友人们的帮助（同时也是现在的伙伴们）。它旨在推动计算机技术与艺术设计，现分为 程序应用／视觉设计／信息安全 三个组别。

我们因为热爱计算机／艺术设计的初衷而走一起，将来无论在哪里，也会继续下去。
