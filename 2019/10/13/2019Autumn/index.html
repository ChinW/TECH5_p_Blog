一、TECH F5VE / TECH5 简介

TECH F5VE（或 TECH5）是由一群热爱计算机技术与艺术设计的学生自主发起的，不隶属于任何组织／部门／机构的技术向组织，其中发起人为Chi。TECH F5VE 于2014年秋成立，其取意为“技术”与“刷新”，即希望我们能够虚心且不断创新。团队从成立至今，始终坚持技术分享，不断进取的初衷，积极参与各类技术交流大会、创新型比赛（Google Developer Day、Hackthon etc.），Follow 最新最炫的技术。目前毕业成员就职经历涵盖阿里、字节跳动、网易、绿盟等一线互联网企业。如果你也是一位来自江大的热爱技术／设计的同学，那么一定不要错过本次的纳新报名哦！

  • 时间 即日~2019年10月20日 周日 12:00
  • 报名入口
  • QQ群：604811424

二、关于宣讲会

大学伊始，对技术充满好奇的你一定有很多疑问：我应该学什么？我以后能做什么？APP，网站，游戏是怎么做出来的……
为了让你的技术之路走得更加平稳，本次宣讲会我们准备了这些内容：

Web 前端 & 后端

  • 我们透过浏览器看到的世界变得越来越漂亮，从最早的白底黑字到现在的各种动画交互效果，前端技术经历了怎样的变化？
  • 双十一的时候，你凌晨守着屏幕，等着灰色的抢购按钮亮起。当你和其它几万人点下抢购按钮的时候，这个小小的按钮背后究竟发生了什么？淘宝的工程师们在服务端做了哪些事情，让抢购这件事情变得更加愉快？
  • Web经历了快三十多年的发展，如今的Web有着空前的发展。如果你对Web技术有兴趣而不知从哪里开始，我认为本次宣讲会是一个不错的出发点。

Python 爬虫

  • “人生苦短，我用Python。”嗯？你问我用Python能干什么？
  • Python是一门功能强大的脚本语言，其丰富的科学计算库使得Python成为了数据科学的首选语言。同时Python也能完成许多很“酷”的事情：自动批量操作文件、爬取音乐/视频、渗透测试等。

Linux系统

  • 《黑客军团》里的主角使用的系统是Kali Linux，事实上在服务器上运行的绝大多数系统都是Linux。Linux有搭建各种高可用服务的技术，是所有业务的“底盘”，工程师们始终关注开源，跟进社区最新动态并力争混迹于此，专注于C、Golang和Java，开发从内核到容器化应用。在本次宣讲会中，我们将从最简单的开始，带你领略命令行的魅力。

游戏

  • 喜欢玩游戏的你有没有想过，有朝一日自己也能开发一款游戏？领略过《风之旅人》，《怪物猎人》，《刺客信条》等大作的奇思妙想之后，你是不是也有种冲动，想要构建一个自己的美丽世界？
  • 做游戏距离我们有多远，一个游戏是如何被开发出来的，如今的游戏行业是否值得进入，你是否适合去做游戏？做游戏需要哪些技能点？Game团队专门制作了一个游戏来为你解答。

全栈技术

  • 我们有幸邀请到了TECH F5VE元老级成员，在最近的秋招中斩获网易互娱SP，腾讯，字节跳动等大厂offer的技术大牛黄毅为大家介绍其它各种技术栈，并和大家分享他在秋招中的经历，以及他的技术学习方法论。

时间地点

关于纳新的一些问题

我能获得什么

  1. 在这里你能找到一群志同道合的人，同样热爱一件事情，让你保持对计算机技术的热爱。
  2. 从别人身上获取你所不知道的知识。
  3. 了解行业动态，技术的更替和相关的比赛事宜。
  4. 由于我们历届的成员已经进入到各个企业／学校内，因此群内会不时拥有内推机会！

我需要做些什么

  1. 学习，技术需要时间去积累，只有不断学习才能尽快成长
  2. 分享，在学到了新东西以后，不要忘记去和大家分享，TECH F5VE的初衷是坚持技术分享、不断进取。

我们看重什么

  1. 向我们展示你的热情，对技术的钻研与执着
  2. 坚持。坚持积累，坚持热爱技术
