2018秋季纳新说明

Oct 8, 2018

注意：该信息已经过期

一、TECH F5VE / TECH5 简介

  • TECH F5VE（或 TECH5）是由一群热爱计算机技术与艺术设计的学生自主发起的，不隶属于任何组织／部门／机构的技术向组织，其中发起人为Chi。TECH F5VE 于2014年秋成立，其取意为“技术”与“刷新”，即希望我们能够虚心且不断创新。团队从成立至今，始终坚持技术分享，不断进取的初衷，积极参与各类技术交流大会、创新型比赛（Google Developer Day、Hackthon etc.），Follow 最新最炫的技术。如果你也是一位来自江大的热爱技术／设计的同学，那么一定不要错过本次的纳新报名哦！
  • 时间 即日~2018年10月14日 周六
  • 报名入口(已过期)

二、一些改变

为了团队和成员未来更好的发展，从今年开始，我们对TECH F5VE的组织架构进行了进一步的划分，从原来 “程序应用／视觉设计／信息安全” 三个组别，划分为以下七个小组：

2.1 移动应用

  • 基于 Android/iOS 系统，上至日常使用的移动应用，下到系统底层的“黑科技”，我们不断发掘移动应用的魅力。
  • 主要方向：原生APP开发、Flutter开发…

2.2 GAME

  • Game组热衷于设计并创造游戏，探索并深入底层。我们致力于深入游戏的每个阶段，引擎是我们创造世界的工具，策划是我们描绘世界的蓝图。在这里，你能尽情制造快乐，收获快乐，享受快乐。
  • 主要方向：计算机图形学、游戏客户端开发…

2.3 Web前端

  • Web有你想要的一切，以通过互联网丰富生活为主要目的，时刻与最新的趋势保持同步。浏览器是我们的世界，在这里我们可以尽情地毁灭和创造。如今我们的舞台拓展到桌面应用（Electron）、移动APP（React Native ），而不是仅仅局限于网页。
  • 主要方向：网站页面开发、WebApp开发、微信公众平台开发…

2.4 Web后端

  • 后端没有前端那般精美绝伦，让人赞叹的画面，但是后端却支撑起了千千万万的前端应用，我们负责编写运行于服务器的代码，这一部分不直接和用户打交道，但它是一个应用程序的核心：处理数据库中千百万条数据、构建高性能、安全稳健的后端程序以应对日益增长的流量和用户。
  • 主要方向：网站后台开发、企业级应用开发、云服务…

2.5 信息安全

  • 这是一个神秘的小组，ErRoR cOdE: 404。
  • 主要方向：网络攻防、渗透…

2.6 视觉设计

  • We need u！！！
  • 主要方向：平面设计、UI设计…

2.7 GEEK

  • Geek组是Geek们的聚集地。在这里，我们设计精巧算法，编写高性能代码，处理海量数据。高性能计算、服务器运维、分布式系统、算法开发、AI、IOT…我们以算法和Linux为根基，致力于探索各种计算机技术。
  • 主要方向：编写高性能的代码、处理海量数据、探索各种高性能中间件…

  1. 线下宣讲会 (时间：2018.10.13 周六下午 14:30、地点：计算机学院 校本部三食堂楼下创新创业孵化基地创咖)，交流会将包括以下内容：
    1. TECH F5VE 介绍
    2. 自由提问环节 (新人向老成员提问)
    3. Sharing (技术分享)
    4. 活动总结
      Attention

    6. 活动时间及地点以纳新群通知为主
    7. 面试前请先进行网络报名
  2. 线下面试 (时间：2018.10.14 周日下午 14:30、地点：计算机学院 校本部三食堂楼下创新创业孵化基地创咖)

四、也许你内心有一些疑问

（1）我进入TECH F5VE之后要做什么？

  1. 学习新的编程语言和使用的工具（语言任意，可以根据学院的教学计划选择，工具包括编译器和git等，所有内容自学）
  2. 尽可能去了解其他成员（更多指老成员）的技能栈；了解自己的兴趣方向并制定学习计划。
  3. 本学期末或下学期，新成员将以小组形式协作完成一个小型应用程序的开发，并进行展示与演讲。

（2）我在这里会得到什么？

  1. 在这里你能找到志同道合的人，同样热爱一件事情，让你保持对计算机技术的热爱。
  2. 从别人身上获取你所不知道的知识
  3. 了解行业动态，技术的更替和相关的比赛事宜
  4. 由于我们历届的成员已经进入到各个企业／学校内，因此群内会不时拥有内推机会！

（3）我要付出什么？

  1. 热情与汗水！💦
  2. Keep Open Mind， Keep in Touch！共享精神是TTECH F5VE强调的事情，学习不是一件孤独的事情，在TECH F5VE，我们要求互相学习。
  3. 时间 + 坚持…

（4）TECH5在纳新的过程中到底看什么？

  • 我们关注新人对新事物的热情与执着，对于技术没有畏惧的心理，也同样关注你是否愿意分享与交流。所有评审结果由组织主观打分和决定，不能保证决定是百分之百得正确，如果最后没有加入，请不要因此放弃对技术的热爱；如果加入了，就继续勇敢地坚持下去。
  • 有一定的抗压能力，具有自主学习心态和能力；富有责任心和创新精神，团队合作能力强。

五、说在加入之前的话

  • TECH5是一个年轻有活力的组织，但是也纯粹出于自发，我们因为热爱一件事物而走在一起，将来不论在何处，也会继续下去，因此希望加入者们能将这一特点发扬光大。
  • 对于布置的题目和任务无法及时完成的，会有一次提醒机会，望珍惜。特殊情况请提前与组织者联系。
  • 长期（指1个月 - 3个月）处于“僵尸”状态，我们默认你并不愿意继续留在TECH5，为了防止误踢，我们会提前询问你。
  • TECH5现在的目标是成为江苏大学内有水平的兴趣爱好组织，提供一个平台给热爱计算机的同学进行交流，我们期待与您一同进步。
  • 组内成员并无任何义务必须回答或帮助其他人，请在提问前善用搜索，对于提供解答的人说声”谢谢”，但是我们推荐大家把经过思考和琢磨的问题拿出来与大家分享与交流，以及在力所能及的范围内进行帮助。
  • 鼓励自由言论，但是不要引起公愤。
