一、TECH F5VE/TECH5 简介

  • TECH F5VE（或 TECH5）是由一群热爱计算机技术与艺术设计的学生自主发起的，不隶属于任何组织／部门／机构的技术向组织，其中发起人为Chi。TECH F5VE 于2014年秋成立，其取意为“技术”与“刷新”，即希望我们能够虚心且不断创新。团队从成立至今，始终坚持技术分享，不断进取的初衷，积极参与各类技术交流大会、创新型比赛（Google Developer Day、Hackthon etc.），Follow 最新最炫的技术。目前毕业成员就职经历涵盖阿里，字节跳动，网易，绿盟等一线互联网企业。如果你也是一位来自江大的热爱技术／设计的同学，那么一定不要错过本次的纳新报名哦！

二、关于宣讲会

大学伊始，对技术充满好奇的你一定有很多疑问：我应该学什么？我以后能做什么？app，网站，游戏是怎么做出来的……

为了让你的技术之路走得更加平稳，这次宣讲会我们准备了这些内容：

Web 前端

  • 我们透过浏览器看到的世界变得越来越漂亮，从一个小小的按钮到各种绚丽的动画效果，我们都可以一窥前端在其中扮演的角色。
  • 屏幕上炫酷的动画，你有想过它是怎么从无到有构成的吗？当然，我们的旅途并不局限于浏览器，在手机APP等其他平台，也能看到前端活跃的身影。
    • 主要方向：网站、APP、客户端搭建

Web 后端

  • 后端虽然没有前端那样炫酷的界面和交互，但是从早期的单机时代，到现在超大规模的互联网时代，几亿人参与的春运，几千亿成交规模的双十一，后端支撑起了千千万万的前端应用，我们负责编写运行于服务器的代码，这一部分不直接和用户打交道，但它是一个应用程序的核心：处理数据库中千百万条数据、构建高性能、安全稳健的后端程序以应对日益增长的流量和用户。

python

  • python是一门功能强大的脚本语言，其简洁、优美的语法特点深受程序员的喜爱， 其丰富的第三方库也使python成为了数据分析的首选，日常工作中也可以写脚本实现自动化办公,可以从网络中批量爬取图片、视频等等，“人生苦短，我用python”。此外、python在人工智能方面拥有着极佳的前景，目前市面上大部分的人工智能的代码都是使用Python 来编写。

游戏

  • 喜欢玩游戏的你有没有想过，有朝一日自己也能开发一款游戏？领略过《风之旅人》，《怪物猎人》，《刺客信条》等大作的奇思妙想之后，你是不是也有种冲动，想要构建一个自己的美丽世界？
  • 在这次的游戏宣讲中，你将会 get 到游戏开发的乐趣。

学长组分享
